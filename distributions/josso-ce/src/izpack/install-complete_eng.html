JOSSO CE Installation Complete!

1. Start JOSSO Community Edition by running the "atricore" command located within the "bin" directory in the target path.

2. You'll be ready to run the Atricore Web Console. In order to launch it, hit the following URL: http://jossoserver:8081/atricore-console.
You should be able to access the server directly from your local system (localhost) as follows by using this URL : http://localhost:8081/atricore-console

3. Finally, sign in using the default credentials: 'admin' as the username and 'admin' as the password.

Use the following to help you get started with JOSSO Community Edition : http://www.josso.org/docs/en-US/html/
